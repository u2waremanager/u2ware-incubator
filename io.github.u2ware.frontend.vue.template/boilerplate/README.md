# Vue CLI
## install 
```
npm install @vue/cli -g
```
## create
```
vue create <app>

cd <app>
```
## plugins
```
vue add vuetify

vue add i18n

vue add router


vue add axios

npm install qs --save

npm install vue-moment --save

```
